# uDC_module


## Pinout

### Encoder connector
1) RED - +5V
2) BLACK - GND
3) WHITE - Encoder channel A
4) YELLOW - Encoder channel B
5) ORANGE - Encoder index
6) GREEN - Analog input

### Limit switch connector
1) RED - GND
2) BLACK - Limit switch A
3) WHITE - GND
4) YELLOW - Limit switch B
5) ORANGE - GND
6) GREEN - Limit switch C
